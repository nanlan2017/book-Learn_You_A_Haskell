“-> 运算符是右结合的” right association :    "从右往左计算"
                f :: a -> b -> c -> d    
                        相当于 f :: a -> (b-> (c->d))
                f a :: b-> (c -> d )      【 f a 是一个两参数、返回一个值的函数】
                     ▇▇▇▇▇▇▇▇▇▇▇▇▇▇ 柯里化 【 f a 是一个函数；只有1个参数；返回一个函数(签名为 c->d) 】 
                f a b :: c -> d       
                f a b c :: d

        @  f 接受3个参数，可以得到d
        @  f 接受两个参数：一个a 类型的值，一个b->c 的函数。  可以得到d
                        f::  a -> (b->c) -> d
                =>      f::  a -> ( (b->c) ->d)
                vs      f::  a -> ( b-> (c->d))
                 这是不一样的！！！
